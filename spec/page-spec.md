# 页面规格文档

## 整体布局

### 单页应用结构
```
┌─────────────────────────────────────┐
│          Header (固定导航)           │
├─────────────────────────────────────┤
│                                     │
│          Hero (首屏区域)             │
│                                     │
├─────────────────────────────────────┤
│                                     │
│          About (关于我)              │
│                                     │
├─────────────────────────────────────┤
│                                     │
│          Projects (项目展示)         │
│                                     │
├─────────────────────────────────────┤
│                                     │
│          Contact (联系方式)          │
│                                     │
├─────────────────────────────────────┤
│          Footer (页脚)               │
└─────────────────────────────────────┘
```

### 响应式布局规则
| 屏幕宽度 | 布局模式 | 项目卡片列数 |
|---------|---------|-------------|
| > 1280px | 桌面端 | 3列 |
| 768px - 1280px | 平板端 | 2列 |
| < 768px | 移动端 | 1列 |

## Hero 区域

### 布局结构
```
┌─────────────────────────────────────┐
│         Hero 区域                    │
│  ┌──────────┐  ┌──────────────────┐ │
│  │  Avatar  │  │ Name & Title     │ │
│  │  (圆形)  │  │ Summary          │ │
│  └──────────┘  │ Social Icons     │ │
│                └──────────────────┘ │
│                                     │
│         Scroll Indicator            │
└─────────────────────────────────────┘
```

### 交互效果
- 页面加载时淡入动画
- 社交图标悬停缩放效果
- 向下滚动提示动画（持续弹跳）
- 点击滚动提示平滑滚动到 About 区域

### 动画参数
- 入场动画：fadeIn + slideUp，持续 0.8s
- 社交图标悬停：scale 1.1，持续 0.3s
- 滚动提示：bounce 动画，无限循环

## 全局交互

### 平滑滚动
- 所有内部链接点击触发平滑滚动
- 滚动行为：easeInOutCubic，持续 0.5s

### 滚动动画
- 使用 Framer Motion 的 useInView 钩子
- 元素进入视口时触发入场动画

### 响应式断点
- 桌面端：> 1280px
- 平板端：768px - 1280px
- 移动端：< 768px

### 性能优化
- 图片懒加载
- 组件懒加载（按需导入）
- 减少重排重绘
